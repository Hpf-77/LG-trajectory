 “LGlight.m”  is the code for describing the x-space wave function of an LG beam .
“xbohm_traj.m” is the code for calculating the Bohm trajectory using the x-Bohm theory. 
“xbohm_pic.m” is the code for drawing the picture after calculating the trajectory using the x-Bohm theory,
